# 🛒 SnapKart

SnapKart is a full-featured e-commerce application built using React Native for the frontend, and a backend powered by Express, Node.js, and MongoDB. The app includes an admin panel for managing products, users, and orders, and integrates the Stripe payment interface for secure and seamless transactions.

## ✨ Features

- **🔐 User Authentication**: Sign up, log in, and manage user profiles.
- **🛍️ Product Management**: Browse products, search, and view product details.
- **🛒 Shopping Cart**: Add products to the cart, update quantities, and proceed to checkout.
- **📦 Order Management**: Place orders and view order history.
- **🛠️ Admin Panel**: Manage products, users, and orders.
- **💳 Payment Integration**: Secure payments using Stripe.

## 🛠️ Tech Stack

- **📱 Frontend**: React Native, Redux
- **🌐 Backend**: Node.js, Express.js
- **🗄️ Database**: MongoDB
- **💳 Payment Gateway**: Stripe

  
## 🛠️ Admin Panel

To access the admin panel, log in with an admin account. The admin panel provides functionality to:

- **📦 Manage Products**: Add, edit, and delete products.
- **👥 Manage Users**: View and manage user information.
- **📝 Manage Orders**: View order details and update order status.
